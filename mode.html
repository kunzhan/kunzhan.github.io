MSE 与模态坍塌

扩散模型 L2 损失的"条件均值陷阱"——公式推导 + 双峰 GMM 交互演示

第一步：MSE 的最优解是条件期望

扩散模型训练时，给定带噪观测 x_t，用 MSE 拟合干净信号 x_0：

θ* = argmin_θ E[ ‖f_θ(x_t, t) − x_0‖² ] 对任意固定 x_t，逐点最小化可得最优解： f_θ*(x_t, t) = E[x_0 | x_t]
▸ 这是最小均方误差（MMSE）的标准结论，与模型结构无关——MSE 就是在学条件均值。

第二步：一对多映射的本质

前向加噪：

x_t = √ᾱ_t · x_0 + √(1−ᾱ_t) · ε, ε ∼ 𝒩(0, I)

当 ᾱ_t 较小（噪声大）时，噪声项主导，多个不同的 x_0 都能产生几乎相同的 x_t。 此时后验分布 p(x_0 | x_t) 变得模糊、多峰。

第三步：具体推导——双峰高斯混合

真实分布
p(x₀) = ½ 𝒩(−μ, σ²) + ½ 𝒩(+μ, σ²)
两个等权的高斯峰，关于原点对称
带噪分布
x_t | x₀ ∼ 𝒩(√α · x₀, (1−α)·I)
令 α = ᾱ_t（信号保留率）

用贝叶斯公式计算后验权重：

p(x_t | x₀=μ_k) · p(x₀=μ_k) w_k = ───────────────────────────────────── Σⱼ p(x_t | x₀=μⱼ) · p(x₀=μⱼ) exp(−(x_t − √α·μ_k)² / (2(1−α))) w_k = ──────────────────────────────────────── Σⱼ exp(−(x_t − √α·μⱼ)² / (2(1−α)))

MSE 最优预测（条件均值）：

E[x₀ | x_t] = w₁(x_t) · (−μ) + w₂(x_t) · (+μ)
关键情形：x_t ≈ 0（两峰正中间）

对称性 → w₁ = w₂ = 0.5，因此：

E[x₀ | x_t≈0] = 0.5·(−μ) + 0.5·(+μ) = 0

但 x₀ = 0 从来不是真实数据——它落在两峰之间的低概率空谷里！
结论（模态坍塌）： MSE 逼迫模型输出多峰分布的加权均值， 这个"均值"可能落在真实数据概率极低的区域。 对图数据而言，即是拓扑上无意义的"模糊结构"。

直觉总结

蓝色曲线（后验）有两个峰。MSE 的最优解是这两个峰的加权平均（黄色虚线）。 当 x_t 处于两峰中间时，这个平均值恰好落在低密度空谷里， 在真实数据分布中概率极低。

把"信号保留率 α"拖到 0.3–0.6、噪声点 x_t 拖到 0 附近， 底部面板会显示 MSE 预测比真实峰值低多少倍——这正是 CFDA 边际正则化的动机： 对预测边际分布 p_θ(x̂₀) 加 Fisher 散度惩罚，让预测样本落在高密度区， 而非被 MSE 拉向无意义的"均值空洞"。
